Stir - React Documentation

Introduction

Stir - Multipurpose Agency React Template. This is created especially for individuals and agencies who provide IT Solutions and Software Services through online. The template contains multiple unique design variations in the home and other pages for the website. Mobile First. Design of the template looks beautiful stunning on all types of screens and devices. Individuals or Agencies who are providing Web, Mobile, eCommerce, AI, Machine Learning or Digital Marketing focused services could love the Stir template for their existing or next business website!

Setup Development Environment

In order to kickstart the development of the web app with Stir, the very first thing you need to do is set up a React development environment.

Following tools are needed to set up a React dev environment:

  • React JS

    • It is recommended to download and install the latest version from the React JS official site https://reactrouter.com/en/main/start/overview.

Preparation

You'll need to install React Router for React installation with router and DOM.

Use a command line tool like PowerShell or Terminal and navigate to the project (Stir) root. This is PowerShell in Windows and Terminal on Mac/Linux.

Note:

This project is set up with the React.js frontend framework, so you have to decide what you do: run with React.js

Let's look at how to set up with React.js :

Just open the terminal inside the project root & run: npm install then move forward one directory such as client cd stir and run npm install again for the client. Then move back one directory with cd .. and run npm run dev. You will see something like the following:

> (your downloaded directory) F:\stir>

> Read on http://localhost:5173 or http://localhost:5174

Compiled successfully!

You can now view stir in the browser.

Local: http://localhost:5173

Just pick the client folder and open the terminal inside it. Run: npm install to install all the dependencies for the Stir React app. After that, run npm start and you will see something like the following:

Starting the development server...

Compiled successfully!

Local: http://localhost:3000

On Your Network: http://192.168.1.6:3000

Now in the browser go to localhost:3000
